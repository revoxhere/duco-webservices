Wrap coins

Duino-Coin is a hybrid currency, meaning that it can be sent to a chain of another network where it will be stored in a decentralized form. You can read more about tokenized versions of Duino-Coin here.
To wrap your coins, simply enter your wallet address, select the destination network and the wrapped amount.

Note: fees may apply: bscDUCO: 500 DUCO, wDUCO: 250 DUCO, maticDUCO: 0 DUCO, celoDUCO: 0 DUCO
Transactions delayed or not arriving? See this document for more info

Warning: make sure you're using the correct wallet and it supports tokens!

Mining key setup

Here you can setup a mining key for additional account protection. After setting it, only miners that have this key in the config file will be able to mine on your account. It's recommended to set this to a strong one to prevent others mining on your account messing your Kolka drops and rewards. Do not share it with anyone.

Note: old miner versions with no mining key support can still use the key authentication. To do so, add the key after a colon in the rig name/id field,
e.g. Home miner becomes Home miner:YOUR-KEY

To disable the mining key authorization, set it to None in the input box below.

Stake coins

Duino-Coins can be frozen for a certain amount of time to earn passive rewards. Staking helps Duino control it's circulation. Currently, you can stake your coins for 21 days to earn up to 1.5% interest rate on them. The minimum stakeable amount is 20 DUCO.
Example: by freezing 1000 DUCO, after three weeks you will receive your initial 1000 DUCO back plus the staked amount - 15 DUCO in this case.

Staked DUCOs get frozen and can not be used while the staking process is active. Staking rewards get sent automatically after the staking period. Staking can't be cancelled.
